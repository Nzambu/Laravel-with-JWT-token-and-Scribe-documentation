    Introduction

    This documentation aims to provide all the information you need to work with our API.

    As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

    Base URL

    http://localhost

    Authenticating requests

    This API is not authenticated.

    Login

    Gain access to the application

    Example request:

    curl -X POST \
    "http://localhost/api/login" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"username":"possimus","password":"esse"}'

    const url = new URL(
    "http://localhost/api/login"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "username": "possimus",
    "password": "esse"
}

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

    Body Parameters

    username  string  
    The registered username.

    The user secret code.

    Refresh token

    Refresh the bearer token when user is logged in and token is about to expire

    Example request:

    curl -X GET \
    -G "http://localhost/api/refresh" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json"
    const url = new URL(
    "http://localhost/api/refresh"
